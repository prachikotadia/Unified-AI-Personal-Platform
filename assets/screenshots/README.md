# Screenshots

This folder contains screenshots of the OmniLife application.

## Required Images

Please add the following screenshot images to this folder:

- `dashboard.png` - Main dashboard view
- `finance-dashboard.png` - Finance module dashboard
- `fitness-dashboard.png` - Fitness module dashboard
- `marketplace.png` - Marketplace module view
- `social.png` - Social module view

## Image Guidelines

- Recommended size: 1200x800 pixels or similar aspect ratio
- Format: PNG or JPG
- File size: Keep under 500KB per image for optimal loading
